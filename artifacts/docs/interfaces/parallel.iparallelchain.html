Interface IParallelChain<TIn, TEnv, TOut>

The parallel chain allows to chain multiple operations before they are executed on a worker.

Type parameters

  • TIn

    the input values created by a generator function

  • TEnv: IEmptyParallelEnvironment

    the type of the environment

  • TOut

    the type of the resulting elements
